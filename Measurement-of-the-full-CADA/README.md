# Measurement of full CADA

### 
This file contains our measurements and runtime evidence for CADA

**Disclaimer**: *All runtime results are based on the `App/attack_app/` source files the corresponding input, which are in our CADA codebase and the corresponding input. All the Measurement results are generated by the source files of `App/` and the corresponding input, which are also in our CADA codebase and the corresponding input.
This repo only presents results supporting RQ3 in the paper.*
## Folders Description
- `Evaluation/`: this folder contains the app's runtime evidence in RQ3.
- `Measurement/`: this folder contains the app's Measurements in RQ3.
- `database_verfy.py`: This file can verify that the runtime evidence of the application conforms to the Measurements.
